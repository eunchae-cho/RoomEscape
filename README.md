# RoomEscape (방탈출 예약 사이트)

  ```
  - 오픈한 방탈출의 내용을 볼 수 있다.
  - 시간에 따른 방탈출을 예약할 수 있다.
  - 방탈출 이용 후 리뷰를 작성할 수 있다.
  - 관리자는 어드민 페이지를 통해 예약된 시간을 볼 수 있다.
  - 예약 상태를 변경/취소하고 탈출 여부를 설정할 수 있다.
  - 룸테마를 등록해 홈페이지에 출력될 수 있도록 한다.
  ```
    
### 📃 기술
  - Java/JSP
  - Spring
  - JavaScript
  - Mybatis
  - MariaDB
